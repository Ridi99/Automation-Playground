# 🎬 Regex Media State Organizer & Auto-Sync (V1 & V2)

**Status:** 📦 Archive (Circa Q1 - Q2 2025)  
**Environment:** Android OS (Termux/Pydroid)  

## 📖 Context
These scripts document the evolution of an early automation pipeline I built to solve media player subtitle desync issues. I built **V1** in March 2025 to learn string manipulation and regex. By July 2025, I refactored the architecture into **V2** to handle edge cases, bad data, and pipeline stability (Idempotency and Fault Tolerance).
